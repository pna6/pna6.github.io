Oscilloscope¶

ADC Trigged¶

Command¶

adc_trig mode level port zip
  • mode (int) : trigger slope and inphase/quadrature trigger.
Modes¶
  Inphase Quadrature
Positive 1 2
Negative 3 4
  • level (int) : trigger level.
Level¶
Minimum Maximum
-4096 4096
  • port (int) : port id, example: 1/2.
  • zip (int) : enable/disable compression.
